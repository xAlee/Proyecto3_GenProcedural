# Proyecto3_GenProcedural

Este proyecto tiene la intención de generar una cantidad de niveles enorme a base de algoritmos de generación, esto mediante
unos datasets iniciales con los algoritmos WFC y Markov implementados en Unity, dejamos ambos repositorios aqui ⤵️

WFC: https://github.com/univDarko/lab-8-wfc

MARKOV: https://github.com/univDarko/lab-10-markov


Con estos algoritmos podemos crear miles de mapas para luego alimentar una NeuronalNetwork y Una GAN, 
estas estan implementadas en google collab, repositorio aqui ⤵️
